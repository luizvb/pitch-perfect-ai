# PitchPerfect AI 🚀

An AI-powered Micro-SaaS that generates hyper-personalized cold emails for sales representatives and startup founders.
Built to solve the pain of writing manual, unscalable, yet personalized cold outreach.

## Features
- **Lightning Fast AI Generation:** Powered by Google's Gemini Flash Lite models.
- **Premium UI/UX:** Built with Next.js 14, TailwindCSS v4, dark mode, and sleek micro-animations.
- **Local History Tracking:** Uses PGlite to persist your generated pitches locally.
- **End-to-End Tested:** Built with Playwright for solid reliability.

## Tech Stack
- Next.js 14 (App Router)
- React
- Tailwind CSS
- PGlite (Database)
- Google GenAI (via OpenRouter)
- Playwright (E2E Tests)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set your `OPENROUTER_API_KEY` in `.env.local`
4. Run the development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)
